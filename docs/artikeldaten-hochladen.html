Format für das Hochladen der CSV Datei

Dateiformat: CSV
Seperator: ;
Encoding: UTF-8
Spalte A: Eine eindeutige ID für das Rückspielen ins ERP-System.
Spalte B: Produkt Information (Motorradmodel, Baujahr etc.).

Wichtig: Die Spaltennamen müssen exakt wie im Beispiel benannt werden. Spalte A „articel_number; Spalte B „input_text“. Außerdem muss die Datei „Semikolon“ separiert sein.
